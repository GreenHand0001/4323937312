# 项目简介

## 核心功能
- 基于YOLOv8的蜜蜂检测系统，支持单类目标检测（bees类）
- 提供7200张蜜蜂数据集及完整标注
- 包含训练好的模型及训练结果
- 支持图片、视频及摄像头实时检测
- 界面功能包括：目标位置显示、目标计数、置信度展示
- 支持批量检测与结果保存
- 可自定义置信度和IOU阈值
- 提供模型选择、标签显示选项

## 适用场景
- 蜜蜂种群研究与监测
- 农业授粉昆虫计数
- 生态保护与生物多样性研究
- 昆虫行为学观察记录
- 养蜂场自动化管理

## 技术特点
- 基于Python3.9开发，依赖opencv-python、PyQt5、ultralytics、torch1.9等库
- 采用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 提供完整UI界面，支持交互式操作
- 文件结构包含：程序源码、UI源文件、图标、数据集及教程文档
- 支持多种输入源，具有良好的扩展性和实用性
